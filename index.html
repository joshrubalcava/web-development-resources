Web Developer Resources

Stop wasting time trying to find web development resources online.

Here you will find great free resources needed for web development!

freeCodeCamp

freeCodeCamp offers a full curriculum with practice exercises so that new developers can learn the basics. They also have many articles containing helpful explanations and/or interesting topics about the tech industry.

MDN Web Docs

MDN Web Docs is one of the largest sources of information on anything related to HTML, CSS, and JavaScript. This website has documentation on just about anything you may need.

Stack Overflow

Git

Git is a free and open source software for version control. It allows developers to track changes in any set of files and is used by developers who are collaborating on a project.

Here are some additional resources to get you started with Frontend Libraries & Frameworks, Backend Development and Deployment

Bootstrap

Bootstrap is one of the most frequently used open-source CSS Frameworks that is directed towards responsive, mobile-first web development.

React

One of the most popular open-source JavaScript Libraries for building user interfaces based on UI components. React was developed and maintained by Meta.

Node.js

A popular transition after learning frontend development. Node.js is an open-source, cross-platform, backend JavaScript runtime environment that runs on a JavaScript Engine and executes JavaScript code outside the web browser.

Netlify

Netlify is a free and powerful choice that includes build, deploy, and serverless backend services foy your web applications and dynamic websites.
